# ☕ Orivaxa Cafe | Premium Coffee Experience 

Welcome to the official repository for **Orivaxa Cafe**, a modern and luxurious landing page designed for coffee enthusiasts. This project focuses on a premium user experience with a warm, caffeine-inspired aesthetic.

---

## 🌟 Project Highlights

* **Premium Visuals:** A sleek, dark-themed design using a rich color palette (Primary Brown, Accent Gold, and Cream).
* **Fully Responsive:** Optimized for all devices, from large desktops to mobile screens 📱.
* **Interactive UI:** Includes hover effects, smooth scrolling, and tactile button animations for a modern feel.
* **Signature Menu:** A beautifully organized grid showcasing various coffee blends with high-quality imagery ☕.
* **About & Heritage:** A dedicated section that tells the story of our beans and brewing process.
* **Integrated Contact System:** A professional contact form and info section for customer inquiries 📧.

---

## 🛠️ Technologies Used

* **HTML5:** For structured and semantic web content.
* **CSS3:** Featuring Custom Variables (CSS Variables), Flexbox, and CSS Grid for a modern layout.
* **Google Fonts:** Utilizing *Playfair Display* for elegant headings and *Poppins* for clean readability.
* **Unsplash API:** High-resolution imagery for a professional look.
