# 📱 Smart Calculator Web App

A modern and user-friendly calculator web application built using **HTML, CSS, and JavaScript**. The project features a unique 3D aesthetic achieved through **Neumorphism UI** design principles.

## 🌟 Project Highlights

* 🎨 **Modern Design:** Utilizes Neumorphism UI to create a stylish, premium, and soft 3D visual experience.
* 📱 **Responsive Layout:** Fully optimized to provide a seamless experience across Desktop, Tablet, and Mobile screens.
* ⚙️ **Advanced Functions:** Supports basic arithmetic along with Percentage (%) calculations and robust "Error Handling" for division by zero.
* 🖱️ **Interactive Effects:** Features real-time shadow transitions upon button clicks, providing a tactile and engaging user interface.

---

## 🛠️ Technologies Used

* **HTML5:** For structured and semantic web markup.
* **CSS3:** For styling, Grid Layouts, and custom shadow effects.
* **JavaScript (ES6):** For core mathematical logic and interactive event handling.
